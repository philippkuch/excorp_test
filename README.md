# excorp_test
Тестовое задание на ETL и аналитику.

Было выполнено:
1. Создан и настроен VDS с Airflow и Grafana в контейнерах;
2. Настроен двухшаговый ELT процесс - сначала получение всех данных с pricempire и сохранение их в JSON, затем обработка конкретного скина AK-47 | Redline (Field-Tested), с транфсормацией данных;
3. Настроено два дашборда метрик качества ELT пайплайна и здоровья данных;
4. Выполнена аналитика данных, включая базовую и бизнес-аналитику, а так же небольшую ML-модель линейной регрессии.
